# Personal Expense Tracker

A simple web-based expense tracker built using HTML, CSS, and Vanilla JavaScript for the Friendsware Solutions Internship Screening Project.

## Features

### Core Features

* Add expenses with a title, amount, and category
* Form validation with inline error messages
* Delete any expense from the table
* Running total updates automatically
* Amounts displayed in PKR format

### Bonus Features

* Filter expenses by category
* Sort expenses by amount (Ascending)
* Sort expenses by amount (Descending)
* Local Storage persistence (expenses remain available after page refresh)

## Categories

* Food
* Transport
* Utilities
* Entertainment
* Other

## How to Use

1. Enter the expense title
2. Enter the expense amount
3. Select a category
4. Click **Add Expense**
5. Use the **Filter Category** dropdown to view specific categories
6. Use **Sort Amount ↑** to sort from lowest to highest amount
7. Use **Sort Amount ↓** to sort from highest to lowest amount
8. Click **Delete** to remove an expense

## Technologies Used

* HTML
* CSS
* Vanilla JavaScript

## Data Persistence

This application uses browser Local Storage to save expenses. Added expenses remain available even after refreshing or reopening the page.
